# FotoWidget

FotoWidget v3.0 — widget slideshow foto untuk Windows.

**Developer:** UZNBT  
**Lisensi:** Gratis, bebas dibagikan  

## Fitur
- Slideshow otomatis (interval 5 detik)
- Transisi: fade
- Urutan: acak (shuffle)
- Tombol kontrol: Zoom+, Zoom−, Lock, Tutup, Ganti Folder, Ingat Folder, Shuffle
- Startup otomatis di Windows

## Cara install
1. Unduh `FotoWidget_Setup.exe`.
2. Jalankan dan ikuti instruksi installer.
3. Pilih lokasi install

## Uninstall
Jalankan Uninstaller dari Control Panel atau dari folder aplikasi. Uninstall akan menghapus semua file aplikasi termasuk konfigurasi.
