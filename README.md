# FotoWidget

FotoWidget v1.0 — widget slideshow foto untuk Windows.

**Developer:** UZNBT  
**Lisensi:** Gratis, bebas dibagikan  
**Bahasa:** Indonesia

## Fitur
- Slideshow otomatis (interval 5 detik)
- Transisi: fade
- Urutan: acak (shuffle)
- Transparansi yang bisa diatur
- Tombol kontrol: Zoom+, Zoom−, Lock, Tutup, Ganti Folder, Ingat Folder, Shuffle
- Startup otomatis di Windows (opsional)
- Konfigurasi disimpan di folder aplikasi

## Installer
Nama file installer: `FotoWidget_Setup.exe`. Unggah installer ke GitHub Releases agar pengguna bisa mengunduh dari halaman release.

## Cara install
1. Unduh `FotoWidget_Setup.exe`.
2. Jalankan dan ikuti instruksi installer.
3. Pilih lokasi install (default: `C:\Program Files\FotoWidget`).

## Uninstall
Jalankan Uninstaller dari Control Panel atau dari folder aplikasi. Uninstall akan menghapus semua file aplikasi termasuk konfigurasi.
